Velocity sensitive electric drum set

Arduino MEGA, Arduino IDE, Piezo electric sensor, virtual drum set software

Velocity sensitive electric drum set

Description

Introduction

Embarking on the fusion of technology and musical innovation, I proudly present my DIY Velocity-Sensitive Electric Drum Set. Crafted with passion and ingenuity, this project utilizes piezoelectric sensors, Arduino, and a robust PVC frame to create a responsive and customizable drumming experience. With features like beat sensitivity and minimal latency, this drum set opens new possibilities for musicians and hobbyists alike.

Components used

Piezoelectric Sensors :
Sensitive transducers converting drum hits into electrical signals.

Arduino Board :
Transforms analog signals from piezos into digital signals for processing.


PVC Frame :
A sturdy frame constructed from PVC pipes, providing support for the drum set.

Virtual MIDI and Drum Software :
Integration with virtual MIDI and drum software, such as EZ Drummer, for a versatile sound library.

Custom Sound System :
A self-built sound system using class ab amplifier for authentic drum sounds.

Features and customization

Velocity Sensitivity :
The drum set's sensitivity to the force of beating allows for nuanced and dynamic drumming, emulating the feel of an acoustic drum set.

Extensive Drum Kit Elements :
Inclusive of snare, kick drums, hi-hat, five toms, and four cymbals, the drum set provides a diverse range of sounds for creative expression.

Quick Responsiveness :
Fine-tuned Arduino code minimizes latency, ensuring instantaneous response to drum hits.

Easily Customizable :
The open-source nature of the project allows for easy customization, enabling musicians to tailor the drum set to their preferences.

Code Modification and Latency Reduction :
By modifying the Arduino code, I achieved optimal performance and reduced latency, ensuring a seamless and immersive drumming experience. This fine-tuning process involved enhancing the communication between piezoelectric sensors and the virtual drum software.

Building a sound system

To complement the electronic drum set, I constructed a custom sound system using class ab amplifier using TDA 2030 ic. This system ensures high-quality audio output, providing an authentic and satisfying drumming experience.
