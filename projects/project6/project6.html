Student Alumni Relations Portal

This endeavor aimed to revolutionize the interaction between students and alumni, creating a seamless platform for networking, mentorship, and career opportunities. Developed from scratch, this project embodies my passion for innovation and problem-solving in the tech space.

Project Scope and Objectives :
The primary goal was to eliminate intermediaries in student-alumni interactions, providing a direct avenue for communication and collaboration. The project scope included designing a responsive website accessible to both external visitors and registered users. Implementing features like profile management, job/internship portals, mentorship platform, real-time communication, notifications, and robust admin functionalities.

Technological Stack :

Backend:
Node.js was chosen for its scalability and real-time capabilities, ensuring smooth communication and data handling.

Database:
MongoDB served as the database solution, offering flexibility and scalability for storing user data securely.

Security:
Employed bcrypt for password encryption, session management for user authentication, and rigorous error handling mechanisms.

Model Training:
Utilized the doc2vec model from Gensim for job and internship recommendations, continuously fine-tuning parameters for accuracy.




Key Features Overview :

Profile Management:
Identifying relevant features and the target variable was crucial for the model's success. To enhance the model's predictive power, I employed one-hot encoding for categorical variables, such as city location. This technique transformed categorical data into a binary matrix, ensuring that the machine learning model could effectively interpret and utilize this information.

Job & Internship Portals:
Curated platforms where users can post and apply for jobs and internships based on personalized recommendations. Job postings include company details, eligibility criteria, job roles, and direct application links.

Mentorship Portal:
Facilitates mentor-student interactions, allowing users to seek advice, ask questions, and receive guidance from experienced alumni. Users can specify preferences for mentors based on location, work domains, and expertise.

Real-time Communication:
Implemented WebSocket for group chats, individual messaging, multimedia sharing, and notifications. Ensured seamless communication across devices, enhancing user engagement and collaboration.

Notification System:
Comprehensive notification system for job updates, new messages, profile visits, mentorship interactions, and administrative alerts. Backflow notification exchange logic optimized notification delivery and user engagement.

Admin & Super Admin Functionalities:
Admin roles include content editing, user management, job/internship oversight, and resolving user-reported issues. Super Admin has overarching control, monitoring admin activities, handling escalated issues, and managing user permissions.




Technical Insights and Innovations:

Model Training and Integration:
Developed custom algorithms for job/internship recommendations, integrating machine learning with backend operations.

WebSocket Implementation:
Leveraged WebSocket for real-time communication, creating chat rooms, multimedia sharing, and efficient message delivery.

Optimization Strategies:
Fine-tuned code for responsiveness, server load management, and computational efficiency, ensuring optimal performance across devices.

Inter-process Communication:
Integrated Python scripts for machine learning tasks, enabling seamless communication between frontend and backend processes.




Personal Journey and Challenges:

Overcame challenges in algorithm complexity, scalability, real-time communication optimization, and data security. Invested approximately four months of intensive development, balancing academic commitments and project milestones. Leveraged resources like ChatGPT, Google Gemini, Stack Overflow, and peer collaboration for problem-solving and knowledge acquisition.


Conclusion and Future Prospects:

This project represents a significant milestone in my journey, showcasing my skills in full-stack development, machine learning integration, and user-centric design. Moving forward, I am eager to explore new technologies, contribute to open-source projects, and continue innovating in the tech industry.
